# Kubernetes Sample

Kubernetesで動くアプリケーションのサンプル集

## 開発環境

**Mac**

[Docker Community Edition for Mac](https://store.docker.com/editions/community/docker-ce-desktop-mac?tab=description)からedge版をダウンロードしてください。
そして、PreferenceからKubernetesを有効にしてください。
